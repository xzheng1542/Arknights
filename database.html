Banners

Arknights is a gacha game, which lets players to spend in-game currencey to revieve random vitural items or characters. For specific characters
it would go through a system on banners that would have a rate-up system where players would use the in-game currency that is built and saved through gameplay
and spend in the banner to try to get the virtual characters they hope for.

Global Server and CN Server are roughly 6 months apart, this means that whatever banner CN Server is experiencing currently, we can expect that Global Server
will get that banner 6 months later.
